# تبلیغات

پنل → «تبلیغات» (یا دکمهٔ 📣 در ربات برای مالک). فیلدها: عنوان/متن، جایگاه
(قبل از فایل، بعد از فایل، پیام شروع)، دکمهٔ لینک‌دار (اختیاری؛ کلیک‌ها از
مسیر `/ad/{id}/click` شمارش و 302 می‌شوند)، پلن هدف (مثلاً فقط `free`)،
سقف نمایش (با پر شدن، خودبه‌خود متوقف می‌شود).

تضمین: مسیر تبلیغ کاملاً best-effort است — هر خطایی در تبلیغ، تحویل فایل را
مختل نمی‌کند.
