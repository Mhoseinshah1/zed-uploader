# رفع اشکال

| علامت | بررسی/راه‌حل |
|---|---|
| ربات پاسخ نمی‌دهد | `BOT_MODE`؟ webhook ⇒ `DOMAIN` معتبر + SSL + وب‌هوک ست شده؛ تست محلی ⇒ `polling` |
| خطای دیتابیس | `docker compose ps` → سرویس db سالم؟ `DATABASE_URL` با `POSTGRES_*` هماهنگ؟ |
| وب‌هوک ست نمی‌شود | دامنه از اینترنت با HTTPS معتبر در دسترس؟ `server_name` در nginx درست؟ |
| حذف خودکار کار نمی‌کند | لاگ worker + سلامت redis |
| پرداخت آنلاین نمایش داده نمی‌شود | پنل → درگاه‌ها: وضعیت باید «آماده» باشد (فعال + مرچنت/کلید) |
| «لایسنس معتبر نیست» | پنل → لایسنس؛ کلید/سرور فعال‌سازی/دورهٔ مدارا را بررسی کنید |
| مایگریشن شکست خورد | خروجی `update.sh` راهنمای rollback و مسیر پشتیبان را می‌دهد |
| پنل باز نمی‌شود | کاربر پنل ساخته‌اید؟ (`python -m app.panel.create_user`) کوکی/HTTPS پشت nginx |

لاگ‌ها: `docker compose logs -f api bot worker`
