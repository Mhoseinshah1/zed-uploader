# راهنمای پنل وب

آدرس: `https://your.domain/panel` — ورود با کاربر پنل (ساخت کاربر:
`docker compose run --rm api python -m app.panel.create_user`).

| صفحه | کار |
|---|---|
| داشبورد | آمار کلی + نسخهٔ نصب‌شده |
| کاربران | مشاهده/مسدودسازی |
| فایل‌ها | جستجو، فیلتر پوشه، جزئیات، گذرواژه، انتقال پوشه، غیرفعال/حذف |
| پوشه‌ها | ساخت/تغییرنام/حذف (فایل‌ها حفظ می‌شوند) |
| بازبینی | تأیید/رد آپلود کاربران |
| گزارش‌ها | رسیدگی به گزارش تخلف (غیرفعال‌سازی فایل / رد گزارش) |
| پرداخت‌ها | تأیید/رد کارت‌به‌کارت (idempotent) |
| درگاه‌ها | پیکربندی مستقل CentralPay / زرین‌پال / زیبال |
| تبلیغات | CRUD + شمارش نمایش/کلیک |
| پلن‌ها | قیمت (تومان/استارز)، مدت، سقف فایل |
| ادمین‌ها | مدیریت ادمین‌های ربات |
| تنظیمات | کارت، پیش‌فرض‌ها، آپلود کاربران، جستجوی عمومی، کانال‌های اجباری |
| متن‌ها | ویرایش متن‌های ربات (پیش‌فرض به‌عنوان placeholder) |
| آمار | نمودارها + خروجی CSV |
| پشتیبان | پشتیبان‌گیری/بازیابی/زمان‌بندی |
| لایسنس | وضعیت/انقضا/اثر انگشت + فعال‌سازی |
| پیام همگانی | ارسال متنی + retry ناموفق‌ها |

امنیت پنل: سشن Redis + کوکی امضاشده، CSRF روی همهٔ POSTها، bcrypt، قفل ورود
(۵ خطا/۱۵ دقیقه)، ثبت audit برای هر اقدام.
